# C++ Vehicle Management System

A beginner-friendly C++ program to manage multiple vehicles using classes and vectors.  
This project demonstrates OOP concepts, including private/protected members, member functions, and external member function definitions.

---

## Features
- Input and display vehicle details: ID, owner name, and type  
- Uses vector to store multiple vehicle objects dynamically  
- Demonstrates the use of external member functions (`show`)  
- Simple console-based interface for data management  

---

## Concepts Used
- Classes & Objects  
- Private and Protected Members  
- Member Functions & External Function Definitions  
- STL Vector for dynamic object storage  
- Iterative input/output  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o vehicle_manager
